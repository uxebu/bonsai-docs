 __list__ List of supported attribute names. In addition to the property names listed for DisplayObject, these are the attribute names you can pass to the attr() method. Note that this property is not available in your code, it's just here for documentation purposes.

Properties
Name Type Description
cap string The shape to be used at the end of open Path items, when they have a stroke. Can be one of 'round', 'square', 'butt'. Default: 'butt'
fillColor string The fill color. Default: black.
fill string Alias for fillColor
fillGradient string The fill gradient. Defaults: nothing
join string The shape to be used at the corners of paths. Can be one of 'miter', 'round', 'bevel'. Default: 'miter'
line string The line color. Default: transparent
strokeWidth number The line width. Default: 0
miterLimit number The miter limit of the stroke. Default: 4
